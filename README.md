# connect-four

## Key Features

-   Detects vertical, horizontal and diagonal victories
-   Uses some CSS and jQuery tricks to make game pieces appear to fall behind the board
-   Option to start new game after victory

## Technologies

-   JavaScript
-   jQuery
-   HTML/CSS
